# Installation

```bash
git clone https://github.com/Rhaima96/composition-api-CRUD-laravel8-vue3.git
```

```bash
composer update
```
```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

```bash
npm install
```

```bash
npm run dev
```

Create data base "compapi"

```bash
php artisan migrate
```

```bash
php artisan serve
```
